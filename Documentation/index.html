1.4  OCR image converter ↑ top

This OCR is tesseract-ocr by Google.


   -(void)recognizeImage:(UIImage *)image completion:(void (^) (NSString* result))completion
    Image converting to string version.


Adding tess data:

  1. If you want add more language you should download it from
    https://code.google.com/p/tesseract-ocr/downloads/list

  2. Drag tess data folder to project. Check -> Create folder references

  3. Open DefaultValues.m -> languageToRecognize -> add language string.
    For example @"eng+fra", @"jpn" whatever you want

  4. It may be problem with build. You have to remove GoogleMobileAds.framework or TesseractOCR.framework and add it again.
